Este problema fue seleccionado ya que vimos desde el primer momento que la solución se podria hallar mediante el uso de Tablas Hash. 

Utilizamos la tabla hash para representar las dependencias entre paginas y la validación de actualizaciones. Por otra parte, se utiliza otra tabla hash para registrar los elementos ya procesados en una actualizacion para verificar si se respeta el orden establecido. 

No se considero ninguna alternativa pues vimos que el problema se podía resolver mediante Tablas Hash y no planteamos ningun otro método.
